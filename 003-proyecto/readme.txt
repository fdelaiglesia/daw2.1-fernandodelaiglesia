Gestor de peliculas donde he añadido las sesiones teniendo solo la funcion de CRUD el administrador del sitio, los usuarios normales solo podrán
ver las listas de las pelicula y las fichas de estas, mientras que el usuario administrador podra añadir portadas, editar todas las opciones de la peliculas y añadir 
y editar nuevos generos.
Además he incluido la funcion de buscar pelicula para que sea mas fácil encontrar las peliculas para el usuario.
El usuario para acceder al CRUD es admin con la contraseña admin.
